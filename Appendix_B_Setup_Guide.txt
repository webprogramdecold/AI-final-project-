Appendix B - Setup Guide (AI Trading Analytics)

1) Install required software
1. Install Python 3.9+.
2. Ensure pip is available: python3 -m pip --version
3. Install Git.
4. Verify Git: git --version

2) Clone the repository
1. Open Terminal.
2. Go to your target folder: cd ~/Downloads
3. Clone the project: git clone https://github.com/webprogramdecold/AI-final-project-.git
4. Enter the project root: cd "AI-final-project-"

3) Create environment and install dependencies
1. Create virtual environment: python3 -m venv .venv
2. Activate on macOS/Linux: source .venv/bin/activate
3. Activate on Windows (PowerShell): .venv\Scripts\Activate.ps1
4. Install Python packages from requirements.txt: pip install -r requirements.txt
5. This installs: flask, ccxt, pandas, numpy, scikit-learn, joblib, requests, mysql-connector-python, werkzeug

4) Project setup before run
1. Move to app folder: cd "FProject/AI trading analytics"
2. Optional clean DB setup: python3 scripts/setup_sqlite.py
3. Optional API key setup: export CMC_API_KEY="your_key_here"

5) Train model (optional)
1. Run training: ./train.sh
2. If you skip this step, pre-trained model files already in the repo are used.

6) Start the app
1. Start server: ./run.sh
2. Open in browser: http://127.0.0.1:5002/

7) Login
1. Use demo account: username testuser
2. Use demo password: password123
3. If login fails, register a new account at: http://127.0.0.1:5002/register
